Setup Instructions

  1. Register at cloud.restheart.com
  2. Create a free service instance (you'll get a URL like https://acac36.eu-central-1-free-1.restheart.com)
  3. Edit config.js and set your service URL in the RESTHEART_SERVICE_URL variable
  4. Refresh this page to load your configured URL
  5. Generate an admin JWT token from the admin webapp
  6. Enter your JWT below and click "Setup Backend"
